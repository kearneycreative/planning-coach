# Planning Coach

A structured goal-setting and planning tool built for clarity, accountability, and long-term thinking.

## Overview

Planning Coach guides users through a 1/2-year planning framework — helping translate big-picture vision into concrete, actionable goals across life and work.

## Features (Planned)

- 1, 2-year goal frameworks
- Annual and quarterly planning worksheets
- Weekly review prompts
- Goal tracking and progress check-ins
- Reflection and retrospective tools
